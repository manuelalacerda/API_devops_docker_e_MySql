## 📦 Tecnologias utilizadas

* Docker
* Docker Compose
* MySQL
* API (backend)

## ▶️ Como executar o projeto

1. Clone o repositório:

```bash
git clone <url-do-repositorio>
cd <nome-do-projeto>
```

2. Suba os containers com Docker:

```bash
docker-compose up -d
```

3. Verifique se os containers estão rodando:

```bash
docker ps
```
